# 社区行为准则

EchoMind 社区希望形成尊重、务实并重视隐私的协作环境。

## 期望行为

- 尊重不同经验、背景和技术观点。
- 技术争议聚焦可复现事实、证据和具体权衡。
- 接受建设性反馈，并清楚区分事实、推断和个人偏好。
- 尊重所有聊天参与者和贡献者的隐私。
- 只在 Issue、PR、测试和截图中使用人工合成数据。

## 不可接受行为

- 骚扰、威胁、侮辱、歧视性攻击或持续的人身针对。
- 未经允许公开他人的聊天记录、身份信息、私人内容或联系方式。
- 发布 API Key、令牌、`.env`、真实数据库或其他凭据。
- 利用项目渠道传播恶意代码、垃圾信息或完整漏洞利用细节。
- 在维护者明确要求停止后继续破坏性互动。

## 执行

维护者可以编辑或删除违规内容，关闭讨论，并临时或永久限制参与。处理时会结合上下文、影响和重复程度，并尽量保护报告者和受影响人员的隐私。

如托管平台提供私密报告功能，请使用该渠道报告行为问题。如果尚未提供，请创建一个不含敏感细节的普通 Issue，请求建立私密联系渠道；不得在公开 Issue 中披露私人聊天或身份信息。

本准则适用于仓库的 Issue、Pull Request、讨论、代码审查以及以 EchoMind 项目名义进行的其他协作空间。
